Document Type Declaration (DOCTYPE)

This declaration defines the document type and version of HTML being used. In this case, it's HTML5. 

            
                <!DOCTYPE html>

HTML Document Structure

The element is the root element of the HTML document. The lang attribute specifies the language of the document. 

            
                <html lang="en">

Head Section

This section is for defining the document's metadata and linking external resources. 

            
                <head>
                    <meta charset="utf-8"> <!-- Specifies the character encoding for the document (UTF-8) -->
                    <title>Product Website</title> <!--Sets the title of the webpage displayed in the browser-->
                    <link rel="stylesheet" href="styles.css">  <!--Links an external CSS file ("styles.css") to style the HTML document-->
                </head>

                <body>
                    
                </body>

JavaScript Script

The <script> element includes JavaScript code that enables smooth scrolling when clicking on anchor links within the document. 

            
                <script>
                    // JavaScript code for smooth scrolling when clicking on anchor links
                </script>

Smooth Scrolling JavaScript

            
                // Select all anchor links with href starting with "#"
                document.querySelectorAll('a[href^="#"]').forEach(anchor => {
                    // Add a click event listener to each anchor link
                    anchor.addEventListener('click', function (e) {
                        // Prevent the default behavior of the anchor link
                        e.preventDefault();
        
                        // Scroll to the target element with the matching ID
                        document.querySelector(this.getAttribute('href')).scrollIntoView({
                            behavior: 'smooth'
                        });
                    });
                });

This code provides smooth scrolling behavior when clicking on anchor links that reference sections within the same HTML document.
It targets anchor links with href attributes starting with "#", assuming they are meant to scroll to elements with corresponding IDs.
The smooth scrolling effect is achieved by using scrollIntoView with the option behavior: 'smooth'.
The e.preventDefault() ensures that clicking on the anchor link doesn't trigger the default navigation behavior.
